# Czarodziejski Dworek — strona przedszkola

Strona WWW niepublicznego przedszkola językowo-muzycznego **„Czarodziejski Dworek"**
(Warszawa, Wola). Statyczna strona w czystym HTML + CSS + JS (bez frameworka), wielostronicowa.

## Podstrony
- `index.html` — strona główna
- `o-nas.html` — o przedszkolu
- `oferta.html` — program i zajęcia
- `wsparcie.html` — wczesne wspomaganie rozwoju i terapia
- `kadra.html` — zespół
- `galeria.html` — galeria zdjęć
- `kontakt.html` — kontakt

## Struktura
```
css/style.css   — wspólne style
js/main.js      — wspólny skrypt (modale kadry, akordeony itp.)
img/            — zdjęcia i grafiki
```

## Uruchomienie lokalne
Otwórz `index.html` w przeglądarce. Strona nie wymaga serwera ani budowania.
