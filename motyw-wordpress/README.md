# Motyw WordPress — Czarodziejski Dworek

Główny deliverable projektu: klasyczny, produkcyjny motyw WordPress odwzorowujący
stronę przedszkola **1:1**. Cały CMS to **natywny WordPress** (`wp-admin`) — bez
własnych paneli. Treści edytowalne, układ stały. Działa na zwykłym hostingu PHP + MySQL.

## Zawartość
- `czarodziejski-dworek/` — właściwy motyw (PHP, CSS, JS, obrazy, import treści `_import/`).

## Instalacja (skrót)
1. Spakuj folder `czarodziejski-dworek/` do ZIP (lub pobierz `czarodziejski-dworek.zip` z **Releases**).
2. `Wygląd → Motywy → Dodaj nowy → Wyślij motyw` → wgraj ZIP → aktywuj.
3. Zainstaluj darmową wtyczkę **Advanced Custom Fields (ACF)**.
4. Zaimportuj treści: `Narzędzia → Importuj → WordPress` → `czarodziejski-dworek/_import/tresci-czarodziejski-dworek.xml`.
